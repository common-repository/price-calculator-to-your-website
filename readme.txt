== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Website price calculator'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Website price calculator'.
2. Upload the 'Website price calculator' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Website price calculator' from your Plugins page.
